# APE 2 - Operaciones con Lenguajes

Aplicación web en Flask para trabajar operaciones sobre lenguajes formales:
- Generación de cadenas
- Pertenencia
- Unión
- Concatenación
- Kleene Star
- Kleene Plus
- Crecimiento del lenguaje

## Requisitos

- Python 3.10 o superior
- pip

Dependencia:
- Flask (importado en app.py)

## Instalación
1. python3 -m venv .venv
2. source .venv/bin/activate
3. pip install Flask
4. python3 app.py
